# Transformer AI Digital Twin

This repository contains a simple Transformer-based AI digital twin project.

## Files

- `app.py` - Main application file.
- `train_model.py` - Training script for the Transformer model.
- `dataset.csv` - Dataset used for model training.
- `transformer_model.pkl` - Serialized trained model.

## Usage

1. Install required dependencies (e.g. `numpy`, `pandas`, `scikit-learn`, `torch`, etc.).
2. Run `train_model.py` to train the model.
3. Use `app.py` to run the application.

## Notes

- Add a `requirements.txt` later for dependency management.
- Ensure the dataset is available in the project root before running training.
